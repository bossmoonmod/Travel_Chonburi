เปิดโปรแกรม XAMPP Control Pane
1. กด Start ที่:
    Apache ✅
    MySQL ✅
2. เข้า URL:
    http://localhost/sriracha_travel/
    หรือ
    http://localhost/sriracha_travel/index.php

เปิดและจัดการฐานข้อมูล (SQL)
1. เข้า URL:
    http://localhost/phpmyadmin

เปิดโปรเจกต์ใน VS Code (เขียนโค้ด)
1. เปิด VS Code
2. ไปที่ File → Open Folder
3. เลือกโฟลเดอร์:
    C:\xampp\htdocs\sriracha_travel
